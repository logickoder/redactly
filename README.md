# Redactly

Provides a simple, secure, client-side tool for users to prepare sensitive WhatsApp chat data for external analysis

## Tech Stack

This Progressive Web App (PWA) is built with:

- **React 19** - UI library
- **Vite** - Build tool and dev server
- **TypeScript** - Type-safe JavaScript
- **Tailwind CSS** - Utility-first CSS framework
- **vite-plugin-pwa** - PWA functionality with Workbox

## Features

- ✅ Progressive Web App (PWA) - Works offline and can be installed
- ✅ Client-side processing - All data stays on your device
- ✅ TypeScript - Type-safe code
- ✅ Tailwind CSS - Modern, responsive UI
- ✅ Service Worker - Offline support and caching

## Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

```bash
# Install dependencies
npm install
```

### Development

```bash
# Start dev server
npm run dev
```

Visit `http://localhost:5173` to see the app running.

### Build

```bash
# Build for production
npm run build
```

The built files will be in the `dist` directory, including:
- Service worker (`sw.js`)
- PWA manifest (`manifest.webmanifest`)
- All assets optimized for production

### Preview Production Build

```bash
# Preview production build locally
npm run preview
```

### Linting

```bash
# Run ESLint
npm run lint
```

## PWA Features

This app includes:
- **Service Worker** - Automatically generated by vite-plugin-pwa
- **Web App Manifest** - Defines app name, icons, and display mode
- **Offline Support** - Works without internet connection after first visit
- **Installable** - Can be installed on desktop and mobile devices

## Security

All data processing happens entirely in your browser. No data is sent to any server, ensuring complete privacy and security for sensitive information.
